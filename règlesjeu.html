Basic rules: The concept of the game is to learn technical vocabulary for the DUT Networks and Telecommunications course.

A 4 x 4 letter board hides a word that you have to find in a given time. A clue will give the translation of the word into French, as soon as the word is found, you are credited with 1 point. The board changes and a new word must be found.

The aim of the game is to get as many points as possible in order to be in the leaderboard.
